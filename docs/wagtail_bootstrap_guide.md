# Wagtail bootstrap для прототипа

Команда `bootstrap_wagtail` создаёт и синхронизирует базовую структуру CMS-раздела.

## Что создаётся

- корневая страница CMS-раздела;
- каталог объявлений;
- каталог свободных редакторских страниц;
- базовые страницы-заготовки:
  - О редакционном разделе
  - Как публиковать материалы
  - Контакты редакции
- стартовое закреплённое объявление.

## Запуск

```bash
uv run python manage.py migrate
uv run python manage.py bootstrap_wagtail
```

## Что ещё делает команда

- настраивает `Site` Wagtail на основе `WAGTAILADMIN_BASE_URL`;
- делает CMS-структуру пригодной к работе без ручной сборки дерева страниц;
- безопасно переиспользует уже существующие страницы, если они были созданы раньше.
